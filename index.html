Bitcoin não é propriedade privada

v. 0.1a

1) Não é um bem fisicamente escasso

Um "Jogo das Palavras" é definido a seguir: dois jogadores devem falar uma palavra válida alternadamente. A cada palavra válida, o placar do jogador é incrementado. Uma palavra inválida incrementa o placar do oponente e termina o jogo. Uma palavra é válida se for única, que comece por "a", está definida em um determinado dicionário e se for dita dentro de dez segundos.

Note que pensar primeiro em uma palavra não a torna propriedade de algum jogador. Isso é evidente se um jogador A anota algumas em uma folha de papel e acidentalmente a deixa cair no chão, quando o outro jogador B olha de relance. O jogador B é livre para usar as palavras que viu escritas, ou seja, o jogador A não pode usar-se de legítima defesa quando aquilo acontecer pois palavras não são escassas, apesar do conjunto das palavras válidas ser limitado.

Se palavras não são propriedade privada, números também não são e tampouco são as chaves privadas utilizadas pelo sistema de Bitcoin, não importando se tal número estivesse anteriormente escrito numa folha de papel, num pendrive ou simplesmente gravado na memória da própria pessoa.

2) Não envolve nenhum tipo de contrato

Um "Bitcoin Baseado no Corpo" (BBC) é definido a seguir: uma população consegue manter um sistema de Bitcoin utilizando apenas seus próprios corpos. Os dados da blockchain são armazenados na memória da própria pessoa, os cálculos, verificações e execussão de protocolos são também realizados na cabeça das pessoas, e finalmente, a transmissão de dados se dá pela fala (grito) entre os participantes. Isso é teoricamente possível se as pessoas tivessem uma ótima memória, muita inteligência cognitiva e gargantas poderosas.

A prova de que o BBC é equivalente ao BTC é dado a seguir: podemos alterar o BBC, onde damos às pessoas a possibilidade de aliviar seu uso e esforço corporal: podem usar papéis para anotar a blockchain, usar calculadoras para agilizarem os cálculos e telefonarem para otimizar a comunicação. Se fizermos mais uma mudança, deixar que salvem a blockchain no HD, deixarem os cálculos para a CPU e se comunicarem pela internet, chegamos ao BTC, apesar da estarmos nos referindo inicialmente ao BBC.
